Funções que mudam estrutura e corpo dos titulos e corpo dos Textos

< Title> :

Serve para colocar um tiluto ou "nome" para sua janela da web

< Div> :

Serve para fazer "sessões" dentro dos codígos os separando dentro do bloco de notas html

< P> :

Serve para adicionar um novo paragrafo na pagina

< H1>, < H2>, < H3>, < H4>, < H5> e < H6> :

São o tamanho das fontes sendo o H1 o maior e o H6 o menor

Em e Strong:

  • Em: Italico
  • Strong: Negrito (Bold)

< BR>:

Usado para fazer uma quebra no texto ou dar espaço de um texto para o outro.

Exemplo:

Samuel e Bia foram ao mercado comprar uva. Chegando lá encontraram Jonas e Ana conversando. Quando Jonas e Ana viram Samuel e Bia falaram em unissono:
< br>
< br>
< br>
< br>Jonas e Ana: OIIIII

Listas: numerada e pontilhada:

Para usarmos dessas listas os codigos são:

Numerada:
    < li> < /li>


Exemplo:
  1. Banana
  2. Uva
  3. Maça
Pontilhada: < ul> < li> < /li> < /ul>

Exemplo:
  • Italom
  • Robert
  • Ana

Imagens e Videos


Imagens

Para colocarmos imagens e videos dentro da nossa página é necessario ter uma pasta onde contenha suas imagens e videos dentro ( se quiser denomine-a como "img"). Depois mova essa pasta para dentro da pasta onde você esta salvando os dados do seu bloco de notas HTML. Feito isso volte para o bloco de notas e coloque o seguinte codigo:

< img src="img/nome de como esta salvo o nome imagem.jpg ou png" alt="como você quer chama-la dentro do bloco">

Videos

Para videos não é muito diferente disso basta fazer o mesmo processo de pasta que foi explicado para imagens (se quiser o nome da pasta de videos pode ser videos KAAKAKA), só o codigo que muda sendo:

< video src="videos/nome de como esta salvo o video.mp4" controls>
